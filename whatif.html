Project Decsription

This project utilized Adobe Illustrator to create vector images. Creating a narrative through speculative fiction, I selected the prompt "The Day the Moon Blinked" having a good idea of what imagery to include. To create my tokens, I utilized Adobe functions such as move, selection, anchor point pen, gradient, symmetry, and ellipse tools.
My challenges with this project were due to my lack of experience using Illustrator and being to conditioned to programs that create rastor images. It took multiple attempts to create my first token, making mistakes like using the pen tool instead of the anchor point pen, being unable to fill in the space with color.


Token #1

My first token utilized gradients, borders, and contrast. I learned how to use anchor points creating this token. My main inspiration was the idea of a false prophet, being glorious yet sinister.



Token #2

My second token utilized the shape and mesh tools. I created gold decorations behind the figure to display the grandeur given to the false prophet.
