Project Decsription

This assignment utilized P5-JS, using coding to create multiple shapes to form two different artworks.
My challenges with this assignment were a result of my poor math skills, making mistakes using the X and Y axis and taking plenty of time to calculate the pixel size of each shape. Ultimately, I didn't struggle too harshly with this assignment, but it was definitely tedious.


Landscape

To create this image, I made a canvas of 600px by 400px and used shapes like circles and rectangles. To create a reflection in the water, I made an ellipse using the colors used for the sun, and reducing the opacity. I also added emojis to liven the scene.



Self Portrait

To create a self portrait, I used shapes like circles, semi-circles, ellipses, rectangles, and lines. I layered shapes on top of each other to create a more realistic effect, such as rectangles on top of the eyes to create eyelids.
