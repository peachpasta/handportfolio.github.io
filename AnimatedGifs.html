Project Decsription

This project focused on creating different types of animated gifs using Photoshop. Photoshop has functions such as timeline animation and stop motion animation, both of which I learned to use for this assignment.
This assignment challenged me greatly, before this semester I've tried to animate before but never had the patience to understand animation programs. After much trial and error and rewatching the tutorial multiple times, it finally clicked! I'm grateful to have learned the basics of animation through Photoshop, I can see myself using these tools for future personal projeccts.


Stop Motion

Utilizing mutliple images, I was able to create a stop motion animation. By stacking images into layers and copying and reversing their order, it's possible to create a stop motion animation!



Frame Animation

Frame animations can be created by copying and pasting multiples of the same image and using a timeline to precisely move the image across the screen. This process creates a perfect looping animation.
