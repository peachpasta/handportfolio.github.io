Artist Statement

Welcome to my portfolio! Here you will see the culmination of everything I've learned about digital artwork so far. My approach to digital media focuses heavily on personal expression, planning, and aesthetics. With each project that we took on this semester, I tried to include pieces of myself within every project, whether that be hobbies, ideas, interests, and personality. Throughout the semester, I’ve learned how to create pixelated images, how to distort an image, how to use masking layers, how to animate, how to code in HTML, how to use P5-JS, and how to use Adobe Illustrator. But more than that, I’ve learned how to be more patient with myself. Not everything will immediately make sense to me, but that doesn’t need to be a barrier to creation. This class has been a great reminder that practice and determination are integral to improvement, and despite the challenges you may face, putting in enough elbow grease will result in gaining new skills and confidence.
